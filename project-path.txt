
Paso 1: Obtención de Datos

Para iniciar mi proyecto, lo primero que debo hacer es obtener un conjunto de datos de películas. Buscaré en sitios web como IMDb
o Kaggle para encontrar conjuntos de datos que incluyan información sobre el título de la película, género, calificación, año de lanzamiento, 
recaudación en taquilla, y otros detalles relevantes.

Paso 2: Preparación y Limpieza de Datos

Una vez que haya conseguido el conjunto de datos, utilizaré la biblioteca Pandas para cargarlo en un DataFrame. Con esto, podré realizar una
exploración inicial de los datos y familiarizarme con su estructura y contenido.
En esta etapa, me aseguraré de limpiar los datos adecuadamente. 
Trataré los valores faltantes y eliminaré cualquier duplicado que pueda afectar el análisis y los resultados.

Paso 3: Análisis Exploratorio de Datos

Con los datos limpios y preparados, utilizaré Pandas y Matplotlib para llevar a cabo el análisis exploratorio. 
Contaré la cantidad de películas en cada género para obtener una visión general de las distribuciones. 
Además, calcularé estadísticas de calificación para entender mejor el rango y la dispersión de las puntuaciones. 
También, exploraré las tendencias a lo largo del tiempo, analizando cómo han evolucionado las películas según el año de lanzamiento.

Paso 4: Visualización de Datos

Aprovecharé la biblioteca Matplotlib para crear gráficos atractivos que muestren los resultados del análisis exploratorio. 
Por ejemplo, crearé un gráfico de barras para presentar la cantidad de películas por género, 
un gráfico de líneas para seguir la evolución de las calificaciones a lo largo de los años, y un diagrama de dispersión 
para visualizar la posible relación entre la calificación y la recaudación en taquilla.
